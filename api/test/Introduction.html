Introduction

Sunday, August 10, 2014 11:32 PM

The goal of this Package is to enable the rapid creation of objects for the purpose of testing.

It's basically a "factory_girl", simplified for use with PHP.

Installing

PHP 5.3+ and Composer are required.

In your composer.json, simply add "league/factory-muffin": "~2.0" to your "require-dev" section:

{
    "require-dev": {
        "league/factory-muffin": "~2.0"
    }
}
